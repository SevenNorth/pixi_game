## 雷电法王皮卡丘

使用2D渲染库为：[PixiJS](https://pixijs.com/)

线上体验地址：[雷电法王皮卡丘](http://game.lovinghlx.cn/)
需要鼠标点击 `start` 或者 `gameover` 旁边的 ![restart](http://sevennorth.lovinghlx.cn/imgbed/restart.png)

--------

### Person
  - 方向键 ←↑→↓， 移动皮卡丘
  - 空格键， 朝皮卡丘面对方向释放闪点

### Monster
    怪物自动追踪皮卡丘
    怪物初始速度为Math.E，≈2.718281828459045
### Food
    食物有自己的分值，会自动消失
    食物的存在时间与分值成反比
